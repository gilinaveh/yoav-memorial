🔎 Phase 1: Smoke Test
Does the page even load? Basic sanity checks before digging deeper.
  • Site loads without errors Critical
    Open the site URL. Open DevTools → Console tab before clicking anything.
    Page renders fully. Hebrew title, navigation, and star background all visible. No red errors in the Console.
  • Hebrew/English toggle works
    Click the "EN" button in the top nav. Then click "עב" to switch back.
    Text switches between Hebrew (RTL) and English (LTR). Layout mirrors correctly.
  • Photos load from Google Drive
    Scroll to the photo gallery section (תמונות).
    Photos appear within a few seconds. No "Drive public load failed" error in console.
  • Navigation anchors work
    Click each nav link: סיפורים, נרות זיכרון, תגובות, תמונות, אודות.
    Page scrolls smoothly to each section.
🕯️ Phase 2: Candles (Firebase)
Testing the real-time candle-lighting feature — the core of what we just migrated.
  • Light a candle — it appears immediately Critical
    Scroll to נרות זיכרון. Enter a test name like "TestCandle1" and submit.
    The candle appears in the candles row within 1-2 seconds. Count goes up by 1.
  • Candle persists after refresh Critical
    Refresh the page (Cmd+R). Scroll back to the candles section.
    "TestCandle1" is still there. Count matches what you had before refresh.
  • Candle appears in Firebase console
    Open Firebase console → Firestore Database → Data tab → candles collection.
    A document with name "TestCandle1" and a createdAt timestamp exists.
  • Real-time sync between two windows
    Open the site in two browser windows side-by-side (use Incognito for one). Light a candle "SyncTest" in window A.
    Candle appears in window B within 2-3 seconds WITHOUT refreshing window B.
  • Empty/invalid candle is rejected
    Try lighting a candle with an empty name (just click submit without typing).
    Nothing is saved. Either a warning appears or the submit button does nothing.
