Memorial Project Roadmap

"To infinity and beyond" — Strengthening Yoav's Legacy

🛡️ Phase 1: Security & Stability
  • Remove Hardcoded Passwords Critical Delete the plain-text passwords from index.html and switch to a secure authentication service like Firebase or Supabase.
  • Implement Real-Time Database Connect the site to a database (Supabase/Firebase) so that comments, stories, and candles are saved permanently across all devices.
✨ Phase 2: Interactive Memory Features
  • "Yoav's Map" Integration Create a visual map with pins at his favorite locations (beaches, hiking trails, etc.) that reveal photos and stories when clicked.
  • Global Candle Heatmap Transform the candle section to show a world map where visitors can see the collective light from every location someone has visited the site.
  • Audio Tribute Section Add a sound gallery for recordings of Yoav’s voice or his favorite music to create a multi-sensory experience.
🚀 Phase 3: Performance & Scalability
  • Cloud Hosting Migration Move the site from local storage to a professional host like Netlify or Vercel for 99.9% uptime and faster loading in Israel.
  • Image Optimization (Cloudinary) Automatically resize high-resolution images so the site remains fast and responsive on mobile phones.
💙 Phase 4: Long-Term Engagement
  • "Memory of the Week" Create a spotlight feature on the home page that rotates through different stories or photos every week.
  • Physical-to-Digital (QR Codes) Create QR codes for physical memorial locations that lead directly to the specific gallery or comment section of the site.
  • Legacy Support Add a section for community impact, such as a donation tracker for causes Yoav cared about (paramedic training, swimming).
