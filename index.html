📖 מדריך Google DriveGoogle Drive Guide
שלב 1: יצירת פרויקט Google Cloud
Step 1: Create Google Cloud Project
  1. פתח console.cloud.google.com
  2. לחץ "New Project" → תן שם (לדוגמה: "yoav-memorial")
  3. ב"APIs & Services" → "Library" → חפש "Google Drive API" → Enable
  1. Open console.cloud.google.com
  2. Click "New Project" → Name it (e.g., "yoav-memorial")
  3. In "APIs & Services" → "Library" → search "Google Drive API" → Enable
שלב 2: יצירת API Key
Step 2: Create API Key
  1. "APIs & Services" → "Credentials" → "+ Create Credentials" → "API key"
  2. לחץ "Restrict Key" → תחת "API restrictions" בחר "Google Drive API"
  3. תחת "Website restrictions" הוסף את כתובת האתר שלך
  4. העתק את ה-API key
  1. "APIs & Services" → "Credentials" → "+ Create Credentials" → "API key"
  2. Click "Restrict Key" → Under "API restrictions" select "Google Drive API"
  3. Under "Website restrictions" add your site's URL
  4. Copy the API key
שלב 3: יצירת OAuth Client ID
Step 3: Create OAuth Client ID
  1. ב-Credentials → "Create Credentials" → "OAuth client ID"
  2. Application type: "Web application"
  3. Authorized JavaScript origins: כתובת האתר שלך
  4. העתק את ה-Client ID
  1. In Credentials → "Create Credentials" → "OAuth client ID"
  2. Application type: "Web application"
  3. Authorized JavaScript origins: your site's URL
  4. Copy the Client ID
שלב 4: יצירת תיקיית Drive
Step 4: Create Drive Folder
  1. פתח drive.google.com
  2. צור תיקיה חדשה בשם "Yoav Memorial"
  3. לחץ ימין → "Share" → הפוך ל-"Anyone with the link can view"
  4. מה-URL העתק את ה-Folder ID (החלק אחרי /folders/)
  1. Open drive.google.com
  2. Create new folder named "Yoav Memorial"
  3. Right-click → "Share" → set to "Anyone with the link can view"
  4. From the URL copy the Folder ID (part after /folders/)
